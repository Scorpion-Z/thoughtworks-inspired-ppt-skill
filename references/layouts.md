# Layouts

Registered page patterns for thoughtworks-inspired-ppt-skill.

Each slide should declare a layout id such as data-layout="T01".

## T01 Cover

Cover page with kicker, main title, subtitle, date, and one abstract visual anchor.

## T02 Executive Summary

Core conclusion page with one strong title and three to five cards.

## T03 Section Divider

Chapter divider page with dark background, short label, and one big statement.

## T04 Insight Statement

Single insight page. Use one large statement and limited supporting text.

## T05 Diagnosis Matrix

Problem diagnosis page. Recommended columns: symptom, root cause, impact, action.

## T06 Capability Framework

Layered capability framework page. Use five to six layers.

## T07 Before After

Current state and target state comparison page.

## T08 Roadmap

Three-stage roadmap page. Recommended stages: pilot, standardize, scale.

## T09 Priority Matrix

Two by two priority page. Axes can be value and difficulty.

## T10 Radar

Capability maturity or technology radar page.

## T11 Data Loop

Closed-loop management page. Use no more than six nodes.

## T12 Closing

Closing page with one final statement and three next actions.

## Recommended sequence

Short deck: T01, T02, T05, T06, T08, T12.

Standard deck: T01, T02, T03, T05, T07, T06, T09, T08, T11, T12.

## Selection guide

Conclusion: T02 or T04.

Problem: T05.

System: T06.

Change: T07.

Plan: T08.

Priority: T09.

Maturity: T10.

Loop: T11.

Closing: T12.
